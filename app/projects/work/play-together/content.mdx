# 项目简介

这是一款剧本杀拼车的小程序，同时面向 B 端和 C 端，借助微信小程序在微信生态内更易传播，以及结合服务号通知更易触达用户的特性，帮助线下剧本杀店在微信私域内，更轻松的拓宽客源，更快速的促成拼团组队，更好的沉淀玩家社群。提供针对剧本杀在线拼车场景，更灵活全面的行业解决方案，以及相比其他平台更有竞争力的付费模式。

# 幕后花絮

从技术上讲，这个项目对于我个人来说是一次全新的挑战，因为在此之前我并未系统的学习过，传统网页开发相关的技术栈（比如 typescript 和 css）。

幸运的是，它给了我相对充裕的时间和自由发挥的空间，让我得以真正的从零到一，完成了整个项目的基础架构和工程化构建（包括但不限于业务组件库，状态管理，页面路由，网络请求和日志监控等）。

很显然，我从之前做过的所有项目的经验中，将最好最精华的部分，全部沉淀在了这个项目中。

虽然前期架构费了点功夫，但磨刀不误砍柴工，后期业务的开发上，它带来了极高的效率，交付质量和稳定性，让我们的团队和用户都感到非常满意。
