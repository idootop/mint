# 项目简介

简单来讲，这是一个 Web3 版本的 Discord。

Opencord 名称的寓意，是指成为连接起 DAO 社区，dApp 开发者 和 Web3 共建者的开放纽带 (**Connecting DAOs, DApps and DAO friends**)。

我们的愿景是打造一个开放的 Web3 去中心化社交平台，帮助 Web3 社区以真正 decentralize 的方式运营和发展，连接更多的 Web3 资源和生态。

# 项目亮点

**1. Web3 原生**

支持使用钱包登录，可选择 ENS 作为用户名，或者使用独特的六边形 NFT 头像。

# 技术浅析

Opencord 整体架构由去中心化层，协议层和客户端应用层 3 部分组成。

import architecture from './images/architecture.webp';

<BaseImage src={architecture.src} alt="Opencord Architecture" />

Opencord 从诞生的那一刻起，就决定要做一个真正开放的去中心化社交平台：它不为任何公司或个人所控制，治理权交给社区（Opencord DAO），并通过渐进式去中心化的 Roadmap 逐步实现整套协议的去中心化。

任何组织和个人都可以在上面搭建自己的社区，畅所欲言。还可以通过插件，以模块化的方式，定制和改造自己的社区，自由连接任何 dApp 和已有的 Web2 生态。
