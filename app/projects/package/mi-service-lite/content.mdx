# 使用方法

```typescript
import { getMiIOT, getMiNA } from 'mi-service-lite';

// 获取 Client 实例
const MiNA = await getMiNA(config);
const MiIOT = await getMiIOT(config);

// 获取米家设备列表
await MiNA?.getDevices();

// 播放文本 TTS
await MiIOT.doAction(5, 1, 'Hello world, 你好！');
```
